# Extension Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

You can create these icons using any image editor or online tool.

For now, you can use placeholder images or create simple icons with the mirror emoji 🪞.

## Quick Icon Generation

You can use online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Or create them programmatically using canvas/image tools.
